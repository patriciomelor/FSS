Ayuda interactiva para configuración HESK

 

» Campos personalizados

Los campos personalizados se pueden usar para recoger información adicional de sus clientes. Puede activar hasta 20 campos personalizados seleccionando SI en la columna HABILITAR. Una vez que un campo está habilitado usted tiene varias opciones para definir.

» TIPO de CAMPO
Elija el tipo de campo. Puede ser una de las siguientes opciones:

  • Campo de texto - un campo normal de texto en una línea (<input type="text">)
  • Caja texto grande - área de texto grande con múltiples líneas (<textarea></textarea>)
  • Selección botones - botón de selección con al menos dos opciones (<input type="radio">)
  • Caja selección - Una caja de selección desplegle con al menos dos opciones (<select></select>)
  • Casilla Verificación - casilla con al menos dos opciones (<input type="checkbox">). Multiples opciones pueden ser elegidas (ticked).

» Requerido (obligatorio)
Si lo marca, el campo será obligatorio, en otro caso será opcional.

» Nombre del campo
Dar el nombre único y personalizado para ser mostrado al lado del campo, como por ejemplo "CODIGO POSTAL".

» Localización (Ubicación)
Elija si el campo personalizado se debe mostrar antes o después del campo "Mensaje" al crear o generar un nuevo ticket de soporte.

» Opciones
Distintas opciones se pueden establecer para cada campo en función del tipo seleccionado:

  • Campo de texto - se puede establecer la longitud máxima a introducir (en caracteres) y el valor por defecto
  • Caja texto grande - se puede establecer el número de filas y columnas (<textarea rows="Y" cols="X">)
  • Botón de opción - lista opciones de botón, uno por línea
  • Caja selección - lista caja selección, uno por línea
  • Casilla verificación - lista opciones a seleccionar, una por línea
