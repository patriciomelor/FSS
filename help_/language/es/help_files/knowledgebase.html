Habilitar base de conocimientos

Utilice este ajuste para activar la funcionalidad de base de conocimientos HESK. Usar este ajuste para tabular la funcionalidad de la base de conocimientos (HESK Knowledgebase) entre SI y NO.

Las opciones disponibles son:

  • SI, habilita la Base de Conocimiento
    Habilita la funcionalidad de Base de Conocimiento en HESK
     
  • SI, use HESK sólo como una Base de Conocimientos (Knowledgebase) (desactivas las opciones de soporte)
    Habilita la Base de Conocimientos (Knowledgebase), pero deshabilita las funciones de generar nuevos tickets en el Centro de Soporte Use esta opción si deseas usar HESK como una Base de Conocimiento y no como una herramienta de soporte.

    Tenga en cuenta que los tickets de soporte pueden generarse por el personal (staff) usando EMAIL PIPING y POP3 FECHING si están activados. Los clientes pueden ver y responder los tickets en ese caso.
     
  • NO, desactivar Base de Conocimientos
    Desactiva la función de Base de Conocimientos en HESK

Editor WYSIWYG (WYSIWYG Editor)

Este opción habilita que "lo que ves es lo que obtienes", del inglés "What You See Is What You Get" editor de texto enriquecido para los artículos de la Base de Conocimiento en vez de un editor simple de texto.

Sugerir artículos de la Base de Conocimiento (Suggest KB articles)

Si se establece en SI, Hesk buscará coincidencias del asunto del mensaje con los artículos de la Base de Conocimiento y recomendará a los clientes leer los artículos relacionados antes de la presentación de un nuevo ticket de soporte.

Habilitar calificación (Enable KB rating)

Si se establece en SI, permitirá que los clientes valoren los artículos de conocimiento como útil o no

Habilitar la búsqueda (Enable KB search)

Al habilitar esta característica mostrará un formulario de búsqueda en la parte superior de la interfaz de cliente en el Centro de Soporte, que les permitirá utilizar el formulario de búsqueda para la Base de Conocimientos.

Puede optar por mostrar un pequeño cuadro de búsqueda (situado en la esquina superior derecha) o grande (más visible, cubre todo el ancho de página).

 

Cerrar ventana

 

Mostrar número vistas (Show article views)

Se establece en SI, para mostrar el número de veces que se ha visto dicho artículo de la Base de Conocimiento al público (el número de vistas siempre es visible desde el panel de control del personal).

Mostrar fecha de artículo (Show article date)

Se establece en SI, para ver la fecha del artículo en la Base de Conocimientos al público (la fecha es siempre visible desde el panel de control del personal).

 

Cerrar ventana

 

Número máximo de resultados por búsqueda (Max search results)

Este es la máximo número de resultados que Hesk devolverá cuando alguien realice una búsqueda en la Base de Conocimientos.

Longitud previsualización artículo (Article preview length)

Longitud máxima (número de caracteres) del contenido del artículo de la Base de Conocimiento, que mostrará en la lista de artículos cuando se navega por categorías.

 

Cerrar ventana

 

Categorías por fila

Número de subcategorías que se mostrarán en la fila de la tabla cuando se navega por la Base de Conocimiento.

Artículos subcategoría (Subcategory articles)

Número de artículos (vista previa) que figuran en al visualizar subcategoría.

 

Cerrar ventana

 

Mostrar artículos populares (Show popular articles)

Establece el número de artículos populares (más visitados) se mostrarán en el Índice Centro de Soporte y en el Índice de la Base de Conocimiento, para ambos sitios. Establecer en 0 para desactivar la lista de artículos populares.

Mostrar últimos artículos (Show latest articles)

Mostrar los últimos artículos (más recientes publicados) en el Índice Centro de Soporte y en el Índice de la Base de Conocimiento, para ambos sitios. Establecer en 0 para desactivar la vista de últimos artículos.

 

Cerrar ventana

 

 

 

 

Artículos relacionados (Related Articles)

Se establece en la cantidad de artículos relacionados que deben aparecer debajo de cada artículo de conocimiento.

Establecer en 0 para desactivar los artículos relacionados
