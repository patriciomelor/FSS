Ayuda interactiva para ajustes Hesk

 

» Ajustes Generales

Título del sitio web

Un título corto de su sitio web, por ejemplo "Mi precioso sitio web". Este nombre será utilizado como el título del enlace que se muestra en la esquina superior izquierda de la interfaz de cliente, para que puedan volver a su inicio.

URL del Sitio

URL de su sitio web, por ejemplo "http://www.tusitioweb.com". Asegúrese de que esto es una URL válida y se inicia por http: // o https: //. Esto será utilizado como la URL del enlace que se podrá ver en la esquina superior izquierda de la interfaz del cliente para que puedan volver al inicio.

Webmaster e-mail

Dirección predeterminada de correo electrónico para el WEBMASTER. Asegúrese de introducir una dirección de correo electrónico válida, puede ser el mismo que el correo electrónico de soporte.

"De (Desde):" email

El campo "De (desde):" es la dirección de correo electrónico que se mostrará como remitente del correo electrónico (Desde: cabecera) en todos los correos electrónicos enviados por HESK. Por lo general, se trata de un correo electrónico que no se lee y no se responde dicho correo. Asegúrese de introducir una dirección de correo electrónico válida, puede ser la mismo que el correo electrónico de soporte, pero se recomienda utilizar una diferente.

"De (desde):" nombre

Este es el nombre que se mostrará como remitente del correo electrónico (De / Desde: cabecera) en todos los correos electrónicos enviados por HESK. Si no hay ningún nombre en el nombre "De (Desde):" de la cabecera, se mostrará la cuenta de correo establecida en el campo "De:".

» Ajustes de idioma

Idioma predeterminado

Si ha instalado varios idiomas para Hesk este será el idioma por defecto. Todos los idiomas correctamente instalados se mostrarán en el cuadro desplegable. Por favor, consulte el archivo readme.html para obtener instrucciones sobre cómo instalar más idiomas. El enlace de "Probar carpeta de idiomas" le ayudará a solucionar problemas con instalaciones de idiomas no operativos.

Si hay varios idiomas habilitados se puede establecer el idioma preferido para cada usuario en la página llamada "Perfil" sección "Información del perfil".

Múltiples idiomas

Al habilitar esta configuración, los usuarios y el personal podrán elegir su idioma preferido de la lista de idiomas instalados. ¡No habilite esta función a menos que proporcione apoyo en todos los idiomas instalados!

» Ajustes Base de Datos (Database settings)

Servidor base de datos (Database host)

Servidor de base de datos, normalmente "localhost" funcionará para la mayoría de las instalaciones. No cambie estos datos al menos que conozca nombre de su servidor MySQL. Pregunte a su proveedor de alojamiento si no está seguro o si usted está teniendo problemas de conexión con la base de datos MySQL.

Nombre de la base de datos (Database name)

Introduzca el nombre de la Base de Datos (MySQL). Usted podrá crear una en el panel de control de su proveedor de alojamiento web (hosting) o preguntar a su proveedor para información adicional.

Usuario de la Base de Datos (Database username)

Introduzca el nombre del usuario de su base de datos MySQL. Debe crear uno y asignarle todos los permisos desde el panel de control de su proveedor web(hosting) o consulte a su proveedor para información adicional.

Contraseña de la Base de Datos (Database password)

La contraseña que utilizará para conectarse a la base de datos MySQL. Cuando usted crea el usuario en el panel de control del proveedor de alojamiento (hosting) debe asginar dicha contraseña, o bien contacte con su proveedor para ampliar información.

Prefijo de las tablas (Table prefix)

Es el prefijo que se añade a todos los nombres de tablas. Si se fija como hesk_tablas la tabla de "usuarios" será llamada "hesk_users" en tu base de datos. Esto es útil si se desea ejecutar múltiples instalaciones de Hesk utilizando una sóla base de datos MySQL, dando a cada instalación un nombre único de prefijo de tabla (hesk 1_, hesk 2_, 3_ hesk, ...)
